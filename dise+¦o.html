Modelo de caja

Es un módulo de CSS que define unas medidas para los elementos de la página web a modo de bloques.

  • Padding: página de relleno entre el contenido y el borde y podemos editarlo poniendo padding en la etiqueta de style dentro de un bloque concreto.
  • Borde: el límite del elemento. Podemos modeificar su grosor, modelo y forma , poniendo border en la etiqueta de style y un bloque concreto.
  • Margen: distancia entre dos elementos. Cambiar la distancia poniendo margen dentro de la etiqueta style y en un bloque concreto.

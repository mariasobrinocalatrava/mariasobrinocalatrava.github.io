PROPIEDADES DE TEXTO CSS María Sobrino

• Generic font family

Significa una familia de fuente general, como podria ser sans-serif o serif.
Este párrafo esta escrito con sans-serif

• Specific font family

Esto es un tipo de letra epecifico dependediendo de cual prefieras poner.
Esto se tiene que poner entre comillas y con la etiqueta style.
Este texto está escrito con 60px.

• Font size px

Esto es para cambiar el tamaño de letra y debemos utilizar la etiqueta style
y por lo tanto su valor esta en pixeles.

• Font size em

Podemos cambiar el tamaño con em en vez de utilizar px.
Por ejemplo, este texto está escrito con una fuente un 150%.

• Font-style

Con el atributo font-style se puede cambiar el estilo de letra a cursiva

• Line-height

Se puede cambiar el atributo entre lineas
utilizando line-height dentro de la etiqueta style

• Text-align

Para alinear un texto a donde tu prefieras ya sea
a la izquierda o derecha o incluso el centro se uliza el atributo text-align dentro de la etiqueta de style

• Text-decoration

Para decorar un texto tienes que utilizar
el atributo text-decoration dentro de la etiqueta style María Sobrino
